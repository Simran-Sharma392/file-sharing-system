# File Sharing System 

## Overview
A role-based file sharing web app where:
- Senders can upload files
- Receivers can view/download files

## Tech Stack
- HTML, CSS (Frontend)
- Node.js, Express (Backend)

## Features
- User Registration & Login
- Role-based access (Sender / Receiver)
- File upload & access control

## How to Run
1. Go to backend folder
2. Run: npm install
3. Start server: node index.js (or your main file)
4. Open frontend using Live Server
